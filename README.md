# PRODIGY_SD_05
!!Web Scraping Project Using Python!!

1. Project Overview:- This Web Scraping project is developed using Python to automatically extract product information from an e-commerce website. The program fetches web
    page content, parses the HTML structure, and collects key details such as product name, price, and rating. The extracted data is then stored in a structured CSV file for      easy analysis and future use.

3. Features:-
    - Extracts product name, price, and rating
    - Scrapes data from an e-commerce website
    - Saves extracted data into a CSV file
    - Simple and beginner-friendly implementation
    - Easy to extend for more pages or data fields

4. Technologies Used:-
     - Programming Language: Python
     - Libraries Used:
             1. requests – to fetch webpage content
             2. BeautifulSoup – to parse HTML
             3. csv – to store data in CSV format

5. How the Program Works:-
     - Sends an HTTP request to the target website.
     - Retrieves the HTML content of the webpage.
     - Parses the HTML using BeautifulSoup.
     - Extracts product details such as name, price, and rating.
     - Saves the extracted data into a CSV file.
